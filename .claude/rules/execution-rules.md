# 执行完整性规则

## 必备规则

1. **编辑 -> 全局检索校验 -> 下一步操作**：修改代码后，必须先通过全局检索校验，再进行后续操作。**禁止未经校验连续多次编辑**。
2. **项目全局检索**：应用修改规则时，不能只检查修改过的文件，必须对**整个项目**进行全局检索。务必确认修改带来的连锁影响。
3. **枚举所有检索变体**：检索时需罗列**中文、英文、缩写、同义词**所有变体关键词进行搜索。禁止只凭想到的单一关键词检索。
4. **重复结构全员检查**：修改循环序号结构（第 1 周 / 第 2 周 / 第 3 周、步骤 1 / 步骤 2 / 步骤 3 等）时，**先整理完整检查清单**，再逐项依次处理。禁止跳过中间条目漏改。
